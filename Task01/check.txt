Чек-лист для функции get_triangle_type:

Позитивные тесты:
1. Стороны: 3, 3, 3 → "equilateral" (равносторонний)
2. Стороны: 3, 3, 4 → "isosceles" (равнобедренный)
3. Стороны: 3, 4, 5 → "nonequilateral" (разносторонний)

Негативные тесты:
1. Стороны: 0, 0, 0 → Исключение (неположительные стороны)
2. Стороны: 1, 2, 3 → Исключение (1+2=3)
3. Стороны: -1, 2, 2 → Исключение (отрицательная сторона)
4. Стороны: 5, 5, 10 → Исключение (5+5=10)
5. Стороны: 1, 1, 3 → Исключение (1+1<3)
