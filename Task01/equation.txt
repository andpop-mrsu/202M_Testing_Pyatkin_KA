Чек-лист для квадратного уравнения ax^2 + bx + c = 0:

1. Два действительных корня:
   a=1, b=-3, c=2
   Ожидаемый результат: [1.0, 2.0]
   Описание: D > 0, корни 1 и 2.

2. Один действительный корень:
   a=1, b=2, c=1
   Ожидаемый результат: [-1.0]
   Описание: D = 0, корень -1.

3. Нет действительных корней:
   a=1, b=0, c=1
   Ожидаемый результат: []
   Описание: D < 0.

4. Линейное уравнение с решением:
   a=0, b=2, c=-4
   Ожидаемый результат: [2.0]
   Описание: a=0, уравнение линейное.

5. Линейное уравнение без решения:
   a=0, b=0, c=5
   Ожидаемый результат: []
   Описание: a=0, b=0, c ≠ 0.

6. Бесконечно много решений:
   a=0, b=0, c=0
   Ожидаемый результат: None
   Описание: Все коэффициенты нули.

7. Линейное уравнение с нулевым корнем:
   a=0, b=5, c=0
   Ожидаемый результат: [0.0]
   Описание: Корень x = 0.
